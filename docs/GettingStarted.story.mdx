# Quick Start

If you want to jump right in with a working React app with `openjscad-react` installed, check out the [react-next-openjscad-starter](https://github/aeksco).

## Install

Install `openjscad-react` with [Yarn](https://classic.yarnpkg.com/en/) or [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm):

```
yarn add openjscad-react
```

```
npm install --save openjscad-react
```

## Compatibility

The `openjscad-react` module is compatible with React v16+ and works with ReactDOM. React Native is not supported at this time.
