# Introduction

The `openjscad-react` package exports an `OpenJscadViewer` React component that can be used for rendering [OpenJSCAD](https://openjscad.org) scripts.

NOTE - this package is currently a work-in-progress and is not yet published for public use.

## Compatibility

<Props of={OpenjscadViewer} />
